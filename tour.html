READ

In this section, users with read-only access can view data and reports. They are able to analyze the information displayed on dashboards, explore datasets, and review system metrics without altering any data.

WRITE NOTES

This section covers the functionality for updating and adding notes. Users with "write notes" access can adjust text annotations and remarks related to specific data entries. This ensures that any clarifications or additional context is captured without modifying core data.

WRITE

In this section, users with full write access can create, edit, or delete records within the system. This level of access is reserved for users with advanced permissions who are responsible for managing and maintaining the data integrity of the SAMS system.
